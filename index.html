DOM - METODOS

  • getElementById

    • Ejemplos :

    cons navigator = document.getElementById("navigator");
    console.log(navigator);

  • getElementByTagName

    • Ejemplos :

    const link = document.getElementByTagName("a");
    console.log(link);

  • querySelector

    • Tambien se aplica a los selectores id , clases , clases con desendencia. Ejemplos :

    const link = document.querySelector('navigator nav ul');
    console.log(link);

  • querySelectorAll

    • Ejemplos :

    const link = document.querySelectorAll("navigator nav ul li");
    console.log(link);

  • SetAttribute

    • Ejemplos :

    const link = document.querySelector('section').SetAttribute('id','section=atribute);
    console.log(link);

  • COMO ACCEDER AL DOM

    • Se puede acceder al DOM por medio de los nodos

    Ejemplo

    const link = document.querySelector('section').SetAttribute('id','section=atribute);
    console.log(link);

  • firstChild - LastChild

    • Ejemplo

    const firsTli = document.querySelector('ul').firstChild;
    console.log(firsTli);

    const firsTli = document.querySelector('li').LastChild;
    console.log(firsTli);

  • nextSibling

    • Se puede acceder al siguiente hermano de la lista.

    Ejemplo

    const firsTli = document.querySelector('ul').firsTli.nextSibling;
    console.log(firsTli);

  • innerHTML VS textContent

    • Ejemplo

    = document.querySelector('.content').innerHTML
    Permite agregar SOLO HTML

    = document.querySelector('.content').textContent='SOLO TEXTO';
    Permite agregar contenido.

  • style

    • Ejemplo

    = document.querySelector('.content').style='color: red';
    = document.querySelector('.content').style='display: none ';

  • SetAttribute y el removeAttribute

    • Ejemplo

    = document.querySelector('section').SetAttribute('id','section=atribute');
    = document.querySelector.removeAttribute('id');

    = document.querySelector('data-testid','prueba=1');
    = document.querySelector.removeAttribute('data-testid');
